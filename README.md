# nextjs-setup-series

## How to install `pnpm` package manager

```bash
curl -fsSL https://get.pnpm.io/install.sh | sh -

# or

npm install -g pnpm
```

## How to create a project

```bash
pnpm create next-app --typescript  <project-name>
```

## How to run

```bash
pnpm lint
pnpm dev
```
